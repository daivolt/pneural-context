# Scientific References — pneural-context

Scientific papers that informed the design of pneural-context, with detailed notes on how each finding maps to specific implementation features.

> **Note:** Full-text PDFs are available locally in this directory but are not tracked in git (too large). Use the DOIs to access papers online. Papers [1–2] are open access; papers [3–5] were obtained via institutional access; papers [6–7] are paywalled.

## Available Papers

| # | Citation | Key Finding Used |
|---|------|----------|-----------------|
| 1 | Ebbinghaus, H. (1913). *Memory: A contribution to experimental psychology* (H. A. Ruger & C. E. Bussenius, Trans.). New York: Teachers College, Columbia University. | Forgetting curve: memory decays exponentially over time unless reinforced. Savings method: relearning is faster than initial learning, proving traces persist even after apparent forgetting. Basis for our decay model (`strength *= 0.95` per sleep cycle) and the boost operation (spaced repetition restores strength). |
| 2 | Murre, J. M. J., & Dros, J. (2015). Replication and analysis of Ebbinghaus' forgetting curve. *PLOS ONE*, 10(7), e0120644. https://doi.org/10.1371/journal.pone.0120644 | Validated Ebbinghaus' original parameters with modern methods. Key finding: forgetting curve shows a "jump" at the 24-hour mark, suggesting a sleep-consolidation benefit. This directly supports our consolidation model where a sleep cycle both decays AND consolidates. Their curve-fitting analysis shows logarithmic and exponential functions both fit well — we use the simpler exponential model. |
| 3 | Buzsáki, G. (1996). The hippocampo-neocortical dialogue. *Cerebral Cortex*, 6(2), 81–92. https://doi.org/10.1093/cercor/6.2.81 | **The architectural blueprint for our 3-tier consolidation system.** Buzsáki describes two modes: "open loop" (awake, theta/gamma oscillations, information processing) and "closed loop" (sleep, sharp wave-ripples, memory consolidation). The hippocampus receives input from neocortex → stores traces during waking → replays during sleep via sharp wave-ripples → progressively transfers to neocortical long-term storage. Our `immediate → consolidated → timeless` pipeline directly mirrors this: `immediate` = hippocampal encoding (open loop), `consolidation` = sharp wave-ripple replay (closed loop), `timeless` = neocortical permanent storage. Also informed our procedural memory design: Buzsáki notes that the "burst discharge" mode during sleep modifies synaptic connectivity — analogous to our reinforcement scores increasing on successful task completion. |
| 4 | Diekelmann, S., & Born, J. (2010). The memory function of sleep. *Nature Reviews Neuroscience*, 11(2), 114–126. https://doi.org/10.1038/nrn2762 | **Directly informed our consolidation timing and tier promotion logic.** Key findings we used: (1) SWS (slow-wave sleep) preferentially consolidates declarative memories — our `run_consolidation()` is the computational analogue of a sleep cycle. (2) "Longer sleep durations yield greater improvements" — our consolidation interval is configurable (`PNEURAL_CONSOLIDATION_INTERVAL`, default 6h) to match natural sleep timing. (3) Explicit encoding is required for sleep-dependent consolidation — we only consolidate memories that have `priority != none` and were explicitly added (not ambient). (4) Sleep occurring soon after learning is more effective — our `immediate` tier captures recent memories for consolidation on the next cycle. (5) Emotional/salient memories get prioritized — our `critical` priority entries are auto-promoted to `timeless` tier, mirroring amygdala's role in prioritizing emotional memory during sleep. |
| 5 | Cepeda, N. J., Pashler, H., Vul, E., Wixted, J. T., & Rohrer, D. (2006). Distributed practice in verbal recall tasks: A review and quantitative synthesis. *Psychological Bulletin*, 132(3), 354–380. https://doi.org/10.1037/0033-2909.132.3.354 | **The meta-analytic foundation for our boost/touch mechanism.** Key findings: (1) Spaced practice produces ~2x better retention than massed practice across 839 comparisons. This justifies our `boost_entry` (strength + 0.3) operation triggered on recall/access rather than simply repeating content. (2) The optimal inter-study interval (ISI) increases with the retention interval — for long-term retention (months), longer gaps between reviews are better. This supports our decay model where entries gradually weaken but can be rescued by a boost at any time. (3) The benefit of spacing holds for both explicit/declarative and implicit/procedural memory — validating our decision to apply boost to all memory types, not just `concept` entries. (4) Their finding that over-spacing can reduce benefit (too-long gaps = forgetting before review) maps to our `archive_threshold = 0.1` — entries that decay too far get archived rather than rescued. |

| # | Citation | Key Finding Used | Status |

| 6 | Wilson, M. A., & McNaughton, B. L. (1994). Reactivation of hippocampal ensemble memories during sleep. *Science*, 265(5172), 676–679. https://doi.org/10.1126/science.8036517 | First direct evidence that hippocampal place cells reactivate during sleep in the same sequences as during waking experience. This is the neurobiological basis for our `run_consolidation()` function: just as hippocampal cells "replay" experiences during sleep to transfer them to neocortex, our system replays recent memory entries to extract insights and promote them to consolidated/timeless tiers. | Paywalled |
| 7 | Wilson, B. A., Baddeley, A. D., & Kapur, N. (1995). Dense amnesia in a professional musician following herpes simplex virus encephalitis. *Journal of Clinical and Experimental Neuropsychology*, 17(5), 668–681. https://doi.org/10.1080/01688639508405157 | Clive Wearing's case: severe anterograde and retrograde amnesia with preserved procedural and emotional memory. Despite a 7-30 second memory span, Wearing could still conduct music, play piano, and experience genuine emotion — he greeted his wife Deborah with joy every time he saw her, even moments after she'd left the room. He filled diaries with entries like "8:31 AM: Now I am really, completely awake" and crossed them out moments later. This demonstrates that memory is not a monolith but a system of dissociable subsystems. This directly motivated our separation of `memory_type` (red/concept/procedural/temporal/relation) and our decision to give procedural entries different decay resistance and to never compress procedural knowledge. | Paywalled |

## How These Papers Map to pneural-context Features

| pneural-context Feature | Scientific Basis | Papers |
|------------------------|-----------------|--------|
| `strength *= 0.95` decay per cycle | Ebbinghaus forgetting curve: exponential decay of unused memories | #1, #2 |
| `boost_entry` (+0.3 strength on access) | Spaced repetition effect: recall strengthens memory | #1, #2, #5 |
| 3-tier memory (immediate → consolidated → timeless) | Hippocampo-neocortical dialogue: hippocampal → cortical transfer | #3, #4 |
| `run_consolidation()` pipeline | Sleep-dependent memory consolidation (SWS replay) | #3, #4, #6 |
| `critical` priority auto-promoted to `timeless` | Amygdala prioritization of emotional/salient memories during sleep | #4 |
| `memory_type` classification (5 types) | Dissociation of memory subsystems (declarative, procedural, emotional) | #7 |
| `procedural_memory` table with `reinforcement_score` | Procedural/basal ganglia memory: learned by repetition, resistant to decay | #7, #3 |
| `archive_threshold = 0.1` (forgotten entries) | Ebbinghaus: savings persist even below conscious recall threshold | #1, #2 |
| Red ink entries: `strength` floor at 0.5 | Emotional/priority memory is more resistant to forgetting | #4, #7 |
| 6h default consolidation/decay intervals | Sleep cycle duration; consolidation during SWS | #4 |
| `briefing` cards assemble relevant context | Analogy: just as Mizuta's notebook anchored her in a dissolving present, briefing cards assemble relevant memory to anchor an LLM session | — (Mizuta, *Ever Vanishing Present*, 2017) |
